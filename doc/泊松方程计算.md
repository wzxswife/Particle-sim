# 一维泊松方程数值求解方法

根据课件《第三讲-静电模型2》的内容，泊松方程的数值求解方法主要包括：

1. **直接解法**（三对角矩阵法）—— 适用于一维有限差分离散后得到的线性方程组  
2. **FFT 方法**（快速傅里叶变换）—— 适用于周期边界条件  
3. **迭代法**（课件中略讲，如 Jacobi、Gauss-Seidel、SOR 等）

以下将详细讲解前两种方法的原理、离散过程，并给出通用的 Python 实现及代码解读。

---

## 一、泊松方程及其离散化

### 1.1 基本方程

一维泊松方程（介电常数 $\varepsilon_0=1$）：

$$
\frac{d^2\phi}{dx^2} = -\rho(x)
$$

### 1.2 有限差分离散

在均匀网格 $x_i = x_0 + i\Delta x$ 上，采用中心差分近似：

$$
\frac{\phi_{i-1} - 2\phi_i + \phi_{i+1}}{\Delta x^2} = -\rho_i
$$

整理得：

$$
\phi_{i-1} - 2\phi_i + \phi_{i+1} = -\Delta x^2 \rho_i
$$

这是一个关于 $\phi_i$ 的线性方程组，系数矩阵为三对角矩阵。

---

## 二、方法一：直接解法（三对角矩阵）

### 2.1 边界条件处理

常见边界条件：
- **Dirichlet 边界**：已知 $\phi_0$ 和 $\phi_{N-1}$，方程组只需求解内部点 $i=1,\dots,N-2$。
- **Neumann 边界**：已知导数，需通过虚拟点或差分公式修改边界方程。

此处以 **Dirichlet 边界**（$\phi_0 = a,\ \phi_{N-1}=b$）为例。  
内部点方程：

$$
\phi_{i-1} - 2\phi_i + \phi_{i+1} = -\Delta x^2 \rho_i,\quad i=1,\dots,N-2
$$

写成矩阵形式 $A \boldsymbol{\phi} = \mathbf{b}$，其中 $A$ 为三对角矩阵：

$$
A = \begin{pmatrix}
-2 & 1 & 0 & \cdots & 0 \\
1 & -2 & 1 & \cdots & 0 \\
0 & 1 & -2 & \cdots & 0 \\
\vdots & \vdots & \vdots & \ddots & 1 \\
0 & 0 & 0 & 1 & -2
\end{pmatrix}
$$

$$
\mathbf{b}_i = -\Delta x^2 \rho_i - \delta_{i,1}\phi_0 - \delta_{i,N-2}\phi_{N-1}
$$

> **注**：边界项已移到右端。

### 2.2 Thomas 算法（追赶法）

由于 $A$ 为严格对角占优的三对角矩阵，可用 Thomas 算法高效求解，复杂度 $O(N)$。

**算法步骤：**

1. 将方程写为 $a_i \phi_{i-1} + b_i \phi_i + c_i \phi_{i+1} = d_i$，其中 $a_i = 1,\ b_i = -2,\ c_i = 1$。

2. **向前消元**：
   $$
   w_i = \frac{c_i}{b_i - a_i w_{i-1}},\quad g_i = \frac{d_i - a_i g_{i-1}}{b_i - a_i w_{i-1}}
   $$
   
   初始 $w_1 = c_1/b_1,\ g_1 = d_1/b_1$。

3. **向后回代**：
   $$
   \phi_i = g_i - w_i \phi_{i+1},\quad i=N-1,\dots,1
   $$

### 2.3 Python 实现

```python
import numpy as np

def poisson_direct(rho, dx, phi0, phiN):
    """
    一维泊松方程直接求解（Dirichlet边界）
    
    Args:
        rho: array, 网格上的电荷密度 ρ[i]
        dx: float, 网格步长
        phi0: float, 左边界电势 φ(0)
        phiN: float, 右边界电势 φ(L)
    
    Returns:
        phi: array, 电势分布
    """
    N = len(rho)
    d = -dx**2 * rho.copy().astype(float)
    
    # 边界条件修正（内部点 1..N-2）
    d[1]   -= phi0
    d[N-2] -= phiN
    
    # 三对角系数
    a = np.ones(N-2)
    b = -2 * np.ones(N-2)
    c = np.ones(N-2)
    
    # Thomas 算法
    w = np.zeros(N-2)
    g = np.zeros(N-2)
    
    # 向前消元
    w[0] = c[0] / b[0]
    g[0] = d[1] / b[0]
    for i in range(1, N-2):
        denom = b[i] - a[i] * w[i-1]
        w[i] = c[i] / denom
        g[i] = (d[i+1] - a[i] * g[i-1]) / denom
    
    # 向后回代
    phi_int = np.zeros(N-2)
    phi_int[-1] = g[-1]
    for i in range(N-4, -1, -1):
        phi_int[i] = g[i] - w[i] * phi_int[i+1]
    
    # 组合边界值
    phi = np.zeros(N)
    phi[0] = phi0
    phi[1:-1] = phi_int
    phi[-1] = phiN
    return phi
```

### 2.4 代码解读

- 函数接收电荷密度 `rho`（长度为 N，每个元素是网格中心点的 ρ 值）。
- 构造右端项 `d`，并考虑 Dirichlet 边界：将已知的 $\phi_0$ 和 $\phi_{N-1}$ 移到右端。
- Thomas 算法中，`a, b, c` 对应内部点方程（共 N-2 个未知数）。
- 向前消元计算 `w` 和 `g`，向后回代得到内部点电势，最后拼接边界值返回。

## 三、方法二：FFT 方法（周期边界条件）

### 3.1 基本原理

对于周期边界条件 $\phi(0) = \phi(L)$，且假设平均电势为零，可在傅里叶空间求解。

泊松方程两边做傅里叶变换（离散傅里叶变换 DFT）：

$$
(-k_n^2) \hat{\phi}_n = -\hat{\rho}_n
$$

其中 $k_n = \frac{2\pi n}{L}$，$n = 0,1,\dots,N-1$。

**解为**：

$$
\hat{\phi}_n = 
\begin{cases}
0, & n=0 \quad (\text{设定平均电势为零})\\[4pt]
\dfrac{\hat{\rho}_n}{k_n^2}, & n\neq 0
\end{cases}
$$

最后逆变换得到 $\phi(x)$。

### 3.2 数值实现步骤

1. 给定网格上的电荷密度 $\rho_i$，计算其 DFT：`rho_hat = np.fft.fft(rho)`
2. 构造波数数组 $k_n$，注意要正确处理 Nyquist 频率
3. 计算 $\hat{\phi}_n$，对 $n=0$ 置 0
4. 逆 FFT 得到电势：`phi = np.fft.ifft(phi_hat).real`

### 3.3 Python 实现

```python
import numpy as np

def poisson_fft_periodic(rho, dx):
    """
    周期边界条件泊松方程求解（平均电势为零）
    
    Args:
        rho: array, 电荷密度（长度 N）
        dx: float, 网格步长
    
    Returns:
        phi: array, 电势分布
    """
    N = len(rho)
    L = N * dx
    
    # 计算傅里叶变换
    rho_hat = np.fft.fft(rho)
    
    # 波数数组
    k = 2 * np.pi * np.fft.fftfreq(N, d=dx)
    
    # 避免除以零
    phi_hat = np.zeros(N, dtype=complex)
    
    # n=0 分量置零（平均电势为零）
    phi_hat[0] = 0.0
    
    # 非零波数求解
    mask = (k != 0)
    phi_hat[mask] = rho_hat[mask] / (k[mask]**2)
    
    # 逆变换取实部
    phi = np.fft.ifft(phi_hat).real
    return phi
```

### 3.4 代码解读

- `np.fft.fftfreq(N, d=dx)` 生成与 FFT 对应的波数序列，范围为 $[-N/2, N/2)$。
- 注意离散傅里叶变换的定义：$\hat{\rho}_n = \sum_{j=0}^{N-1} \rho_j e^{-i 2\pi j n / N}$，这恰好是 `np.fft.fft` 的结果。
- 由于周期边界条件，方程中的 $-k^2$ 在 $k=0$ 时奇异，通过人为设定 $\hat{\phi}_0=0$ 消除了常数偏移。
- 最后取实部（理论上应为实数，但数值误差会产生微小虚部）。

## 四、两种方法的比较

| 特性 | 直接法（三对角） | FFT 法 |
|------|----------------|--------|
| 边界条件 | Dirichlet / Neumann / 混合 | 周期边界 |
| 计算复杂度 | $O(N)$ | $O(N \log N)$ |
| 内存需求 | $O(N)$ | $O(N)$ |
| 实现难度 | 中等（需处理边界） | 简单（几行代码） |
| 精度 | 二阶差分精度 | 谱精度（对光滑函数指数收敛） |

> **选择建议**：对于无界或开放边界问题，常采用 Dirichlet 边界且网格足够大；对于循环对称问题，周期边界 + FFT 非常高效。

## 五、完整示例

下面给出一个完整脚本，演示两种方法在相同网格、相同电荷密度下的应用，并与理论解（库仑定律）对比。

```python
import numpy as np
import matplotlib.pyplot as plt

# 参数
L = 100.0
N = 101
dx = L / (N-1)
x = np.linspace(0, L, N)

# 点电荷位置及电量
q1, x1 = 1.0, 23.6
q2, x2 = -1.0, 63.3

# 一阶权重法分配电荷到网格
rho = np.zeros(N)

def deposit(q, xp):
    i = int(np.floor(xp / dx))
    if 0 <= i < N-1:
        w_left = 1.0 - (xp - i*dx)/dx
        w_right = (xp - i*dx)/dx
        rho[i] += q * w_left / dx
        rho[i+1] += q * w_right / dx

deposit(q1, x1)
deposit(q2, x2)

# 方法1: 直接法 (Dirichlet 边界 φ=0)
phi_dir = poisson_direct(rho, dx, 0.0, 0.0)

# 方法2: FFT 周期边界
phi_fft = poisson_fft_periodic(rho, dx)

# 计算电场: E = -dφ/dx
E_dir = -np.gradient(phi_dir, dx)
E_fft = -np.gradient(phi_fft, dx)

# 库仑定律解析电场
def E_coulomb(x, q, x0):
    r = x - x0
    return q / (4*np.pi) * r / (np.abs(r)**3 + 1e-12)

E_exact = E_coulomb(x, q1, x1) + E_coulomb(x, q2, x2)

# 绘图
plt.figure(figsize=(12, 8))

plt.subplot(2, 1, 1)
plt.plot(x, phi_dir, label='Direct (Dirichlet)')
plt.plot(x, phi_fft, '--', label='FFT (Periodic)')
plt.xlabel('x')
plt.ylabel('Potential φ')
plt.legend()
plt.grid()

plt.subplot(2, 1, 2)
plt.plot(x, E_exact, 'k-', linewidth=2, label='Exact (Coulomb)')
plt.plot(x, E_dir, 'r--', label='Direct')
plt.plot(x, E_fft, 'b:', label='FFT')
plt.xlabel('x')
plt.ylabel('Electric field E')
plt.legend()
plt.grid()

plt.tight_layout()
plt.show()
```

### 5.1 结果分析

| 方法 | 特点 |
|------|------|
| **直接法 (Dirichlet)** | 在边界处强制 φ=0，导致边界附近电场偏离解析解 |
| **FFT 法 (周期边界)** | 假设左右边界电势相等，对非周期问题会产生虚假环绕效应 |

两种数值方法在远离边界的区域均能较好逼近库仑电场，但在边界附近各有不同的系统误差。

---

## 六、总结

本讲介绍了两种实用的一维泊松方程数值求解方法：

- **三对角直接法**：处理 Dirichlet 或 Neumann 边界，通过追赶法高效求解，适合大多数非周期问题。
- **FFT 谱方法**：专用于周期边界，具有谱精度且实现简洁，在等离子体物理（如 PIC 模拟中周期边界条件）中广泛应用。

对于复杂边界或变系数问题，还可考虑迭代法（如 SOR），此处不再展开。
